# Contributing to Isahc

Thank you for taking the time to contribute! Please be aware of these guidelines.

- If you have a general question or need help using Isahc, please use the [Discussions forum](https://github.com/sagebind/isahc/discussions). The issue tracker is for reporting bugs, planning potential features, and tracking the development process.
- When reporting a bug, search existing issues in the [issue tracker](https://github.com/sagebind/isahc/issues) first to verify if the bug has already been reported before opening a new issue. We do not close issues due to age; if the problem persists, then the affiliated issue will remain open.
- All contributions must follow the current [AI Usage Policy](AI_POLICY.md).
