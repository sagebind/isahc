Trait Selector 

Source 
pub trait Selector { }
Expand description

Types implementing this trait can be used as a condition to select a network interface to use for outgoing connections based on some criteria.

You can combine multiple selectors in some cases using tuples to apply multiple selectors, such as selecting an interface by both its name and IP address. All selectors in the tuple must match the interface to be selected.

This trait is sealed and cannot be implemented outside of this crate.
